# Trainer Booking Bot (Telegram)

## Быстрый старт
1) Установите Python 3.11+
2) Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3) Создайте `.env` рядом с `bot.py` (можно на основе `.env.example`) и заполните:
   - `BOT_TOKEN`
   - `ADMIN_IDS` (через запятую, без пробелов)
4) Запустите:
   ```bash
   python bot.py
   ```

## Первый запуск (админ)
- Откройте бота → `/start`
- Нажмите **Админ меню**
- Создайте **Группу**
- Создайте **Пригласительную ссылку** для группы
- Разошлите ссылку ученикам: `t.me/<bot>?start=g_<token>`
